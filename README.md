# Deep Galaxy — Visual Architecture Explorer

An open-source static analysis tool that parses a codebase and visualizes its architecture as an interactive force graph — a cosmic map of your code.

## Packages

| Package | Description |
|---|---|
| `packages/scanner` | Node.js/TypeScript CLI — parses `.vue` and `.php` files, outputs `constellation.json` |
| `packages/visualizer` | Vue 3 + D3.js frontend — renders the interactive force graph |

## constellation.json

The schema contract between scanner and visualizer. See `constellation.json` at the root for an example.
